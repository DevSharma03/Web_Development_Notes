## HTML tags ##
$ Overview $
- html tags comes in pair: opening and closing
- Some tags are self-closing
- tags instruct the browser how to render enclosed content


$ Non-Self Closing tags $
<tagname attribute1 = "value1" attribute2 = "value2"></tagname>
- <tagname>: opening tag
- </tagname>: closing tag
- attribute = "value" : modify the tags behaviour(apperiance)

$ Self Closing tags $
<tagname attribute1 = "value1" attribute2 = "value2" />
- <tagname>: opening tag
- attribute = "value" : modify the tags behaviour(apperiance)

### Functions in JS ###
- A reusable block of code that performs a specific task. 
steps to create and use function 
1. define the function (declared)
2. write our code in the function 
3. call/use the function (function invocation)

eg.
let a = 10;
let b = 20;
function Add(){
    result = a+b;
    console.log(result);
}
Add();
a = 20; 
Add();
b = 90;
Add();


$$ Return KeyWord $$
- when we use return keyword it doesnt provide us the output directly we need to use console.log function to print the output.
eg.
function Add(a, b){
    return a+b;
}
console.log(Add(15, 5));


$$ Function Expression $$
eg.
const addNumber = function(a, b){
    let result = a+b;
    return result;
}
let x = addNumber(10, 5);
console.log(x);
alert(x);
